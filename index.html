COVID-19 Detection Using CT-Scans

Machine Learning

Jan 1, 2023

This project explores how deep learning can assist in identifying COVID-19 infections from CT-scan images. The aim was to build a prototype system that classifies scans as 'COVID-positive' or 'Normal', enabling faster screening and supporting healthcare workers during the pandemic.

Tech Stack

Python, TensorFlow/Keras, OpenCV, NumPy, Matplotlib, Pandas, Jupyter Notebook.

Features

  • Custom CNN trained on CT-scan datasets
  • Preprocessing with normalization, resizing, augmentation
  • Experimented with ResNet vs custom CNN

UI & UX Enhancements

  • Visualization of predictions
  • Clear output results (COVID / Non-COVID)

Developer Experience

  • Modular code organization
  • Metric logging
  • Rapid prototyping with notebooks

Results & Insights

  • Achieved ~80% accuracy
  • Data augmentation was critical
  • Dataset diversity influenced outcomes

Getting Started

  1. Collect CT datasets
  2. Preprocess images
  3. Train CNN models
  4. Evaluate results
  5. Prototype with Streamlit

Lessons Learned

  • Data imbalance impacts models
  • Hyperparameter tuning is crucial
  • Deployment needs more validation

Acknowledgments

Open datasets, TensorFlow/Keras community, AI in healthcare research inspiration.
